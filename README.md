# 🔭 Poliscope

Le scope politique français — suivez l'Assemblée Nationale en temps réel.

Recherche des députés, explore les scrutins, suis les votes, compare les élus.

## Stack

- **Frontend:** Vite + Vanilla JS + Tailwind CSS
- **Données:** API Assemblée Nationale (data.nan)
- **Déploiement:** Vercel / GitHub Pages

## Dev

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
npm run preview
```
